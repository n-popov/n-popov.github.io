Задание по третьему семинару

Лекционные задания

  1. (8 б.) Пройдите тест по третьей лекции.
  2. (5 б.) Решите первую задачу «MyEngine» из контеста по третьей лекции.
  3. (7 б.) Решите вторую задачу «Хранилище» из контеста по третьей лекции.
  4. (10 б.) Решите третью задачу «Приключенцы 0.2» из контеста по третьей лекции.

Контрольные вопросы

  1. (5 б.) В чём разница между открытым и закрытым наследованием?
  2. (5 б.) Какие операторы приведения Вы знаете? Разъясните, что они из себя представляют и каковы отличия между ними.
  3. (5 б.) Для чего нужны ключевые слова virtual, override и final?
  4. (5 б.) Что такое абстрактный базовый класс, чисто виртуальная (pure virtual) функция и как они связаны?
  5. (5 б.) Для чего применяется ключевое слово friend? Кто может быть другом класса?
  6. (5 б.) Какие действия с Git Вы можете назвать? Объясните своими словами, что из себя представляют репозиторий и коммит.

Задачи

  1. (50 б.) Реализуйте иерархию классов геометрических фигур, используя абстрактный базовый класс и виртуальные методы. Включите в иерархию квадрат, прямоугольник, треугольник, круг и эллипс. Корректно реализуйте методы вычисления площади и периметра. Добавьте метод, возвращающий количество углов. Реализуйте функции angles_amount, perimeter и square, принимающие указатель на базовый класс.

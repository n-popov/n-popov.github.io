Задание по шестому семинару

Лекционные задания

  1. (15 б.) Пройдите тест по шестой лекции.
  2. (8 б.) Решите первую задачу из контеста по шестой лекции.
  3. (12 б.) Решите вторую задачу из контеста по шестой лекции.
  4. (15 б.) Решите третью задачу из контеста по шестой лекции.
  5. (25 б.) Решите четвёртую задачу из контеста по шестой лекции.

Контрольные вопросы

  1. (10 б.) Что такое функциональный объект? В чём его преимущества или недостатки по сравнению с функцией?
  2. (10 б.) Как устроены лямбда-функции?
  3. (10 б.) Для чего применяется класс std::vector? Какие методы std::vector Вы можете назвать?
  4. (10 б.) Что такое нетиповой аргумент шаблона, пакет параметров шаблона?

Задачи

  1. (25 б.) Реализуйте шаблонную функцию, возвращающую void, первым аргументом которой передаётся функция, последующими — её аргументы. Количество аргументов не известно! Функция должна вызывать полученную функцию с переданными ей аргументами.
  2. (30 б.) Реализуйте шаблонный функциональный объект, вычисляющий факториал с кэшированием результатов (это означает, что если ранее факториал от некоторого числа был уже найден, то он должен быть немедленно возвращён из этого объекта, и никаких вычислений производиться не должно).

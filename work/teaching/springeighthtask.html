Задание по восьмому семинару

Лекционные задания

  1. (12 б.) Решите первую задачу из контеста по восьмой лекции.
  2. (12 б.) Решите вторую задачу из контеста по восьмой лекции.
  3. (13 б.) Решите третью задачу из контеста по восьмой лекции.
  4. (13 б.) Решите четвёртую задачу из контеста по восьмой лекции.
  5. (25 б.) Решите пятую задачу из контеста по восьмой лекции.

Контрольные вопросы

  1. (10 б.) Расскажите о следующих итераторах: std::back_inserter, std::istream_iterator, std::ostream_iterator. Для чего они нужны, в каких случаях применяются?
  2. (10 б.) Что лежит в основе std::set и std::map, std::unordered_set и std::unordered_map? В чём фундаментальная разница между устройством этих контейнеров?
  3. (5 б.) Какие математические объекты могут быть сопоставлены контейнерам std::set и std::map?
  4. (5 б.) Почему при инициализации std::array необходимо задавать размер?

Задачи

Комментарий: все задачи следует выполнить в одном файле. Алгоритмы необходимо применить во всех задачах, кроме 2-й и 12-й.
  1. (10 б.) Создайте array некоторого размера N, где N — натуральное число по Вашему вкусу. Заполните массив случайными числами, используя алгоритм std::generate из библиотеки <algorithm>. В алгоритм следует передать привычный диапазон и лямбда-функцию без аргументов, захватывающую генератор снаружи и возвращающую сгенерированное число.
  2. (10 б.) Создайте пустой list и запишите туда последовательность частичных сумм array, используя алгоритм std::partial_sum из библиотеки <numeric>. В алгоритм следует передать привычный диапазон и специальный итератор std::back_inserter на list.
  3. (10 б.) Выведите получившиеся значения в консоль, используя std::copy и std::ostream_iterator.
  4. (10 б.) Установите соответствие между множеством значений получившегося list-а и множеством их чётностей, создав и заполнив std::map<int, bool>, где true означает, что число чётное.

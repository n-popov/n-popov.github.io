Задание по седьмому семинару

Лекционные задания

  1. (10 б.) Решите первую задачу из контеста по седьмой лекции.
  2. (10 б.) Решите вторую задачу из контеста по седьмой лекции.
  3. (15 б.) Решите третью задачу из контеста по седьмой лекции.

Контрольные вопросы

  1. (25 б.) Перечислите алгоритмы стандартной библиотеки, которые Вы знаете, очень кратко пояснив, для чего применяется каждый из них.
  2. (10 б.) Для чего применяются итераторы? Какие операции с итераторами Вы можете назвать?
  3. (10 б.) Что такое концепция диапазонов и почему она используется в стандартной библиотеке?
  4. (10 б.) Как применять библиотеку random для генерации случайных чисел?
  5. (5 б.) Какой контейнер поддерживает быструю вставку элементов в начало?

Задачи

Комментарий: все задачи следует выполнить в одном файле. Алгоритмы необходимо применить во всех задачах, кроме 2-й и 12-й.
  1. (7 б.) Создайте std::deque, заполнив её числами от 1 до 10 по возрастанию.
  2. (7 б.) Добавьте в начало ещё несколько случайных чисел (в том же диапазоне).
  3. (7 б.) Скопируйте все данные в пустой std::vector.
  4. (7 б.) Отсортируйте его по убыванию.
  5. (7 б.) Перемешайте элементы deque случайным образом.
  6. (7 б.) Вычислите скалярное произведение vector-а и deque.
  7. (7 б.) Определите количество чисел, больших 4, в любом из контейнеров.
  8. (7 б.) Найдите сумму элементов vector-а.
  9. (7 б.) Найдите произведение элементов deque.
  10. (7 б.) Удалите из vector все чётные числа, большие 3.
  11. (7 б.) Удалите из deque все четвёрки.
  12. (7 б.) Выведите vector в консоль с помощью цикла for через итераторы.
  13. (7 б.) Выведите deque в консоль с помощью алгоритма for_each.
  14. (7 б.) Определите минимальный и максимальный элементы в этих контейнерах.

Задание по четвёртому семинару

Лекционные задания

  1. (8 б.) Пройдите тест по четвёртой лекции.
  2. (5 б.) Решите первую задачу «Хищник» из контеста по четвёртой лекции.
  3. (7 б.) Решите вторую задачу «Конвой» из контеста по четвёртой лекции.
  4. (10 б.) Решите третью задачу «Приключенцы 0.3» из контеста по четвёртой лекции.

Контрольные вопросы

  1. (5 б.) Каковы отличия между ссылкой и указателем?
  2. (5 б.) Как и в каких случаях применяется ключевое слово const?
  3. (5 б.) Для чего применяются шаблоны?
  4. (10 б.) Что такое CRTP? Дайте подробный комментарий.
  5. (10 б.) Какой паттерн способствует реализации идиомы RAII? Каким образом он устроен?

Задачи

  1. (15 б.) Расставьте ключевые слова const и осуществите передачу по ссылке (или константной ссылке) там, где это необходимо в реализации классов обыкновенной дроби и иерархии геометрических фигур.
  2. (30 б.) Реализуйте иерархию геометрических фигур через паттерн CRTP. Унаследуйте все фигуры от одного базового класса.
  3. (20 б.) Создайте вектор, в который положите функции square и perimeter, сделав их шаблонными. В цикле осуществите вызов каждой из этих функций для некоторых фигур.

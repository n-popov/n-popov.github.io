Задание по пятому семинару

Лекционные задания

  1. (16 б.) Пройдите тест по пятой лекции.
  2. (10 б.) Решите первую задачу «Студенты» из контеста по пятой лекции.
  3. (14 б.) Решите вторую задачу «2d-вектор» из контеста по пятой лекции.
  4. (20 б.) Решите третью задачу «nd-вектор» из контеста по пятой лекции.

Контрольные вопросы

  1. (10 б.) Какие методы класса std::string Вы можете назвать?
  2. (10 б.) Что Вы можете сказать о семантике перемещения в С++? Какая функция применяется для её реализации?
  3. (10 б.) Расскажите о возможностях библиотеки filesystem.
  4. (10 б.) С помощью каких классов осуществляется ввод и вывод в файлы в C++?
  5. (10 б.) Как можно пройтись по вектору без использования численных индексов?
  6. (10 б.) С помощью каких функций осуществляются преобразования строк в чисел, чисел в строки? Перевод символов между верхним и нижним регистрами?

Задачи

  1. (70 б.) Напишите программу, осуществляющую преобразование и сортировку слов, прочитанных из файла и запись их в другие файлы в соответствии со следующим алгоритмом преобразования:
    1. Удостоверьтесь, что всё слово находится в нижнем регистре. Если нет, приведите. (Осуществите итерацию с помощью range-based for).
    2. Осуществите поиск буквы "c". Если буква найдена, замените её на заглавную и отправьте слово в файл "c.txt".
    3. Если буква "c" не найдена, проверьте количество символов. Если оно чётное, запишите слово в "even.txt", если же нет — в "odd.txt". Перед записью добавьте в конец строки символ "_".
    Указанные файлы должны быть автоматически созданы программой в папке "processed", которая также должна быть создана в результате работы программы (не вручную). Файл с исходными строками создаётся вручную и может лежать в любом удобном Вам месте.

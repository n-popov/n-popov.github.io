Реализуйте параллельный алгоритм суммирования parallel_accumulate через механизм будущих результатов std::future, распараллеливая на аппаратное количество потоков hardware_concurrency.
Осуществите измерение времени работы параллельного алгоритма при разных размерах входных данных, сравните со стандартной версией std::accumulate.
Предложите порог размера входного контейнера, при котором распараллеливание не требуется: при данных меньшей длины быстрее будет вызывать стандартную версию алгоритма. Доработайте алгоритм с учётом этого нюанса.

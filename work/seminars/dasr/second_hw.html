Упражнения:

  1. (70 б.) Решите одну из классических задач на словари. Считайте с клавиатуры N чисел и определите количество повторений каждой из них. Выведите на экран пары <число-вхождения>, отсортированные по возрастанию числа. Подсказка: если вы обращаетесь по несуществующему ключу, то ошибки не будет. Будет сгенерирована новая пара ключ-значение со значением по умолчанию. Например, если у вас пустая std::map<int, int> your_map, и вы делаете std::cout << your_map[42], то выведется 0. Если сделать your_map[322] += 5, то по этому ключу будет лежать значение 0 + 5 = 5. Реализуйте вывод map-ы с помощью оператора <<.

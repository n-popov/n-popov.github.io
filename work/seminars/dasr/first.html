Краткая справка по первому семинару

  1. C++ — мультипарадигменный, статически типизированный, компилируемый.
  2. Компиляция — «перевод» с человеко-читаемого языка на машинный.
  3. Функция main — точка входа в программу.
  4. Стандартное пространство имён — существует; using namespace std; — запрещено (плохая практика). Но на лекции так делают :(
  5. Точка с запятой — в конце каждой строки.
  6. Форматирование — красивое.
  7. Основные типы данных: int, unsigned int, double, bool, char. Их модификации: short int, long int, long long int; float, long double. Важный тип данных: size_t.
  8. Классы потокового ввода-вывода: std::cin, std::cout, операторы потокового ввода-вывода: >>, <<.
  9. Операторы — в основном интуитивно понятные. Нужно помнить: деление интов с помощью оператора «/» выполняется нацело!
  10. Условный оператор if (условие) {..} else if {..} else {..} может содержать от 0 до N блоков else if и/или else, где N — натуральное число.
  11. Циклы: следует использовать только цикл for(выражение-инициализация; условие; инкремент(декремент)) {..}.
